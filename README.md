# AI Escape Room Game (Unity)

## Overview
A 3D escape room game built in Unity using C#. Features AI enemy behaviour with NavMesh pathfinding and state-based logic.

## Features
- AI enemy with patrol and chase behaviour
- Player interaction system
- Game state and level progression
- UI and scene management

## Technologies Used
- Unity
- C#
- NavMesh

## My Role
Designed and developed the full game including AI, gameplay systems, and UI.
